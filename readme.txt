Weather Clothes Picker
By Ari Perez

####Information####
Modules Used:
-requests
-BeautifulSoup

What is this?
This is a simple application which allows you to enter your country and city, and the program will outputthe current weather, and suggested clothing appropriate for the weather

How To Run:
1. Ensure you have the required modules
2. Run the program in IDLE or through command line (python getWeather.py)
3. Enter your Country and City
